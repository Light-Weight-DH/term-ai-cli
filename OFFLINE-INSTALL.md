# 번들 tarball 설치 안내

`term-ai` 릴리스에는 실행에 필요한 의존성을 함께 묶은 번들 tarball이 포함됩니다.  
이 tarball을 쓰면, 대상 환경에서 `npm install`로 외부 패키지를 다시 내려받지 않아도 됩니다.

다만 패키지 자체를 설치하는 단계는 필요합니다.

## 언제 쓰는가

- 인터넷이 막힌 환경
- `npm install`이 정책상 불가능한 환경
- 배포본만 전달하고 바로 실행해야 하는 환경

## 사용 방법

릴리스 자산으로 제공된 `.tgz` 파일을 복사한 뒤 설치합니다.

```bash
npm install -g ./term-ai-0.3.0.tgz
```

로컬 프로젝트에 넣고 싶으면:

```bash
npm install ./term-ai-0.3.0.tgz
```

설치가 끝나면 일반 패키지와 동일하게 실행합니다.

```bash
term-ai
```

## 참고

- 번들 tarball은 의존성을 함께 포함하지만, 패키지 설치 명령 자체는 필요합니다.
- `npm install` 없이도 되는 것은 "외부 의존성 재다운로드"가 아니라 "패키지 자체를 tarball에서 설치"하는 흐름입니다.
- 릴리스 버전과 tarball 이름은 같이 맞추는 것이 좋습니다. 예: `term-ai-0.3.0.tgz`
